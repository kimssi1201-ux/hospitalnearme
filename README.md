# TeenClimate Action

청소년이 기후 위기를 배우고 AI 맞춤형 챌린지, GPS 운동 측정, 칼로리 계산, 기후 뉴스 브리핑, 학교 프로젝트, 실천 기록으로 직접 행동하도록 돕는 교육형 기후 행동 플랫폼입니다.

## 주요 페이지

- `index.html`: AI 코치, 챌린지, GPS 운동 측정, 칼로리 계산, 기후 뉴스, 7일 플랜, 실천 일지, 학교 프로젝트
- `news.js`: 기후 뉴스 RSS 호출과 주제별 브리핑 대체 콘텐츠
- `news.css`: 기후 뉴스 카드와 브리핑 레이아웃
- `movement.js`: GPS 운동 측정과 칼로리/탄소 절감량 계산
- `movement.css`: 운동 측정 섹션 스타일
- `articles.html`: 청소년 기후 행동 글 20개
- `resources.html`: 수업/동아리용 자료실
- `about.html`: 사이트 소개와 운영 원칙
- `contact.html`: 문의와 제안 안내
- `privacy.html`: 개인정보 처리방침과 광고 안내
